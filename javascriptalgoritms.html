Binary search

Branch and bound method

A common algorithmic technique for finding optimal solutions for various optimization problems, especially in discrete optimization and combinatorial optimization. Buchberger's Algorithm In computational algebraic geometry and commutative algebra, Buchberger's algorithm may be a method of remodeling a specific set of generators of a polynomial ideal into a Gröbner basis with reference to a monomial order. This can be seen as a generalization of the Euclidean algorithm for univariate gcd computation and Gaussian elimination for linear systems.

Data compression

Data compression or source coding is that the process of encoding information using fewer bits (or other communication units) than an unencoded representation uses employing a particular encoding scheme

Diffie-Hellman Key Exchange

An encryption protocol that allows two parties without prior knowledge of each other to jointly establish a shared private key over an insecure communication channel. This key can be used to encrypt subsequent communications using symmetric key cryptography.

Dijkstra's algorithm

An algorithm that solves the single-source shortest path problem of directed graphs with non-negative edge weights.

Dynamic programming

Dynamic programming is a method for reducing the execution time of algorithms that demonstrate overlapping sub-problems and optimal substructure characteristics, as described below.

Euclidean algorithm

An algorithm that determines the greatest common divisor (gcd) of two integers. It is one of the oldest known algorithms since it first appeared in Euclid's Elements around 300 BC. This algorithm does not need to factor two integers.

Expected Value Maximization Algorithm (EM Training)

In statistical calculations, the Expectation-Maximization (EM) algorithm is an algorithm for locating the utmost likelihood estimates of the parameters of a probabilistic model whose model depends on unobserved latent variables. The EM alternates between performing an expected step of calculating the expected value of a latent variable and a maximizing step of calculating the maximum likelihood estimate for a given parameter and setting the latent variable to the expected value.

Fast Fourier Transform (FFT)

An efficient algorithm for computing the Discrete Fourier Transform (DFT) and vice versa. FFTs are very important for a variety of applications, from digital signal processing to solving partial differential equations to algorithms that quickly multiply large integers.

The steepest descent method

The steepest descent method is an optimization algorithm that approaches the minimum value of a function by performing steps proportional to the negative value of the gradient (or approximate gradient) of the function at the current point. Instead, taking steps proportional to the gradient approaches the maximum value of the function. This procedure is known as gradient climbing.

Hash

A function for summarizing or probabilistically identifying data. This usually means applying a formula to the data to produce a string that is probably more or less unique to that data. The string is much shorter than the original data, but can be used to uniquely identify it.

Heap (heapsort)

In computer science, the heap is a special tree-based data structure. The heap is a popular data structure in many applications. Heapsort, selection algorithm (finds any kth element of minimum, maximum, or both, median, or sublinear time), graph algorithm.

Maximum flow

The maximum flow problem is finding a legitimate flow through the most important flow network. it's sometimes defined as finding the worth of such a flow. the utmost flow problem are often seen as a special case of more complex network flow problems. Maximum flow is related to cutting the network by the maximum flow min-cut theorem. The Ford-Fulkerson algorithm calculates the utmost flow during a flow network.

Merge sort

A sorting algorithm for sorting lists (or other data structures that can only be accessed in order, such as file streams) in a specified order.
